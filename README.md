# Riyas Siddikk - Portfolio Website

A modern, high-performance personal portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Curtain Landing**: Animated hero overlay with typewriter effect
- **Custom Cursor**: Spotlight/fluid cursor effect
- **Smooth Animations**: Scroll-triggered animations using Framer Motion
- **Responsive Design**: Fully responsive for Mobile, Tablet, and Desktop
- **Dark Mode**: Tech-inspired dark theme with neon cyan/blue accents

## Tech Stack

- Next.js 14+
- React 18
- Tailwind CSS
- Framer Motion
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for Vercel deployment. Simply push to your repository and connect it to Vercel.
